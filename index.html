TinyMCE 4.0.1 forfür OXID eShop CE 4.7

TinyMCE is a platform independent web based Javascript HTML WYSIWYG editor control released as Open Source under LGPL.

TinyMCE has the ability to convert HTML TEXTAREA fields or other HTML elements to editor instances.

Here you will finde more info and exsamples

TinyMCE ist ein open source webbasierter Javascript HTML WYSIWYG Editor, veröffentlich unter der LGPL Lizenz.

TinyMCE konvertiert einfache HTML Textarea-Felder oder anbdere HTML Elemente zu Editor-Instanzen.

Weitere Infos und Beispiele finden Sie hier:

http://www.tinymce.com/

Installation:

  • download the newest version:
    https://github.com/vanilla-thunder/hdi-tinymce/archive/master.zip
  • unzip the archive
  • upload the content of the "copy_this" folder into the shop root directory
  • log in your shop backend and activate the module in the module list
  • you can change some editor settings in the module settings
  • neuste Version runterladen:
    https://github.com/vanilla-thunder/hdi-tinymce/archive/master.zip
  • Archiv entpacken
  • Den Inhalt von "copy_this" in den Hauptordner des Shops hochladen
  • In das Shop backend einloggen und das Modul aktivieren
  • Sie können einige Einstellungen des TinyMCE Editors in den Moduleinstellungen ändern
